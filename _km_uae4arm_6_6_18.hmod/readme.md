=== UAE4ARM Core for RetroArch ===

Module adds support for Commodore Amiga

Available executables and arguments to run Core:
- /bin/amiga <rom> <clover_args>
- /bin/uae <rom> <clover_args>
- /bin/uae4arm <rom> <clover_args>
