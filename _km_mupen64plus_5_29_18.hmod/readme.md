=== Mupen64Plus Core for RetroArch ===

Module adds support for Nintendo 64

Available executables and arguments to run Core:
- /bin/mupen <rom> <clover_args>
- /bin/mupen64plus <rom> <clover_args>
- /bin/n64 <rom> <clover_args>
